## Utiliser le Wake-on-LAN avec l’application Free et une Freebox Pop

Le **Wake-on-LAN (WoL)** permet de réveiller à distance un appareil compatible, comme un PC ou un NAS, depuis l’application mobile **Free** avec une **Freebox Pop**.

Pour que cette fonction soit disponible, il faut d’abord activer l’option **Proxy Wake on LAN** dans la Freebox.

## Activer l’option sur la Freebox Pop

Dans **Freebox OS** :

1. ouvrez **Configuration avancée** ;
2. allez dans **Configuration** ;
3. activez **Proxy Wake on LAN**.

## Réveiller un appareil depuis l’application mobile

Dans l’application **Free** :

1. ouvrez **Espace abonné** ;
2. allez dans **Appareils** ;
3. sélectionnez l’appareil que vous voulez réveiller ;
4. appuyez sur l’encart **Réveiller cet appareil (WoL)**.

## Conditions pour que cela fonctionne

- l’appareil doit être **compatible Wake-on-LAN** ;
- le WoL doit être activé sur l’appareil ;
- l’option **Proxy Wake on LAN** doit être activée sur la Freebox Pop.
